## 钉钉通知功能

### 状态：✅ 已完成

### 背景

 当 Claude-Code 远程执行需要用户输入时（如审批确认），系统已有两种通知方式：
 1. Web 浏览器通知 - useLocalNotification.ts 已实现 ✅
 2. Web Push 通知 - PushService 已实现 ✅

 用户需要新增第三种通知方式：钉钉群通知（通过 Webhook），以便在手机不在身边时也能及时收到提醒。
